# Data Governance

## Data Categories
- Public external data: Ulbrich public product/spec docs.
- Public ML dataset: UCI Steel Plates Faults.
- Synthetic internal data: quote history, policies, pricing/material rules.

## Guardrails
- No real internal customer/ERP data is used.
- Synthetic files are labeled clearly in content headers.
- Claims from quote assistant must map to citations or deterministic rules.
