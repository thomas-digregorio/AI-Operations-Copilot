# Demo Script

## Story 1: Quote Assistant
1. Submit quote request for thin 316L foil with cert requirements.
2. Show rule validation (missing fields, escalation flags, warnings).
3. Generate structured quote draft with citations.
4. Ask follow-up retrieval question and display cited passages.

## Story 2: Steel Defect Classifier
1. Train model from local UCI dataset.
2. Score single input and batch CSV.
3. Display confidence, class, and local feature explanation.
4. Show drift summary against training baseline.
